## Creación del servidor
Se crea el servidor con una IP y un puerto específicos. Este va a escuchar.
Las conexiones entrantes de los clientes.

## Conexión del cliente
El cliente se conecta al servidor utilizando la IP y el puerto específico.
Una vez conectado, se creará una instancia que representará a ese cliente.
Lado del servidor (se crea una para cada cliente que se conecte).

## Comunicación
Una vez conectados, la PC cliente puede enviar paquetes de datos a través de sockets TCP al servidor.
